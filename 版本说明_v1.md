# DustDB v1 版本说明

软件名称：DustDB v1  
中文说明：滞尘数据库检索与下载工具  
第一数据库维护者：杨威，2023级硕士研究生  
邮箱：yangw1663@gmail.com

## 1. 版本功能

1. 数据库路径不预设，由用户在运行时选择汇总表、光谱文件夹和导出位置。
2. 汇总表中的全部表头字段均可作为筛选字段。
3. 同一字段内支持多选值，多选值之间按“任一满足”处理。
4. 多个字段之间支持 `同时满足 AND` 和 `满足任一 OR` 两种逻辑。
5. 光谱文件不作为筛选列，仅通过“同时下载所选数据的光谱 txt”复选框控制是否复制。
6. 支持将筛选结果另存为 CSV 和 XLSX。
7. 自动生成缺失光谱清单和导出日志。

## 2. 数据访问约束

DustDB v1 只执行查询、筛选、复制和另存操作，不修改原始数据库。导出结果写入用户指定的输出目录。

## 3. 适用输入

1. 汇总表：`滞尘数据汇总.xlsx`。
2. 光谱目录：存放以 `标识码.txt` 命名的光谱文本文件。
3. 主键字段：`标识码`。

## 4. 输出内容

1. 筛选后的样本属性表。
2. 可选导出的光谱 txt 文件。
3. 缺失光谱清单。
4. 导出日志。
